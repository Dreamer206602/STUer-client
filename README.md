# 简介

STUer是一个面向汕大学生的校园客户端，它主要有五个功能模块，分别为：拼车、约、转让、树洞以及校内。
STUer的后台使用Severlet + JDBC + MySql的组合进行开发，没有使用什么框架，部署在新浪云平台。
项目所使用的所有图片资源，基本来源于：[http://iconfont.cn/](http://iconfont.cn/)
